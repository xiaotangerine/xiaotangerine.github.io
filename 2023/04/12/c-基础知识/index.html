c#基础知识

2023/4/12

1.变量规则

不能以数字开头、不能以程序关键字命名、不能有特殊符号卡头(除下划线 _ )、驼峰命名法、区分大小写

2.变量类型和使用规则

● int 整数类型 取值范围：最大2147483647;最小-2147483648
● double 小数类型 取值范围：最大 1.79769e+308;最小-1.79769e+308
● string 字符串 “” // 双引号 可以为空
● char 字符 ‘ ‘ // 不能为空但是可以有空字符串必须为单引号 ‘ ‘
● decimal 金钱类型
● bool 我们用bool类型来描述对或者错。
● true 对
● false 错
先声明、在赋值、最后使用、必须以字母开头，后面可以跟任意字母，数字，下划线

3.局部变量

在方法中，我们声明的所有变量都叫做局部变量 。局部变量的作用域就是当前方法

成员变量

在类中，我们声明的变量叫做成员变量。成员变量的作用域当前类，所谓作用域指的就是，我们能够使用这个变量的范围。

4.初始化变量

变量是结构或类的成员时，可以不用显示初始化，默认值为变量类型的默认值default(t)
变量是局部变量时，必须初始化后才能使用，否则编译出错

//定义时初始化
int x=10;
Console.WriteLine(x);
或者：

//定义和初始化分开
int x;
x = 10;
Console.WriteLine(x);

5.弱类型

弱类型必须定义时就要初始化
弱类型一但经过初始化后，类型即确定。后续不能在赋予它其它不能存在隐式转换规则的类型值
static void Main(string[] args)
{
var a = “Hello,World”;
//string类型和整型不能隐式转换，因此不能给a赋值为整型
//a = 10;
//获取变量a的实际类型
Type t= a.GetType();
Console.WriteLine(t);
}

6.常量

常量必须在声明时初始化，后续将不能在修改其值
常量值在编译时进行计算，因此从一个变量中提取值来初始化
常量总是隐式静态的，因此不允许使用static来修饰
const int a=100;
引用类型
存储在托管堆中（managed heap）
它存储的是对值的引用，这类类型是通过对值的引用来复制，修改第一个变量值会影响第二个变量的值
CLR会跟踪引用变量的可访问性进行垃圾回收
